## Khi gửi yêu cầu hỗ trợ

Để hỗ trợ xử lý sự cố nhanh nhất, vui lòng chuẩn bị các thông tin sau khi liên hệ:
1. **Mã lỗi hoặc Triệu chứng cụ thể** (vd: Agent mất kết nối, Alert flap...).
2. **Logs liên quan** (cả phía Agent và Server Master).
3. **Môi trường bị ảnh hưởng** (UAT, Production, Staging).
4. **Địa chỉ Ingress URL & Ingest API Key** đang sử dụng.
